## Change these first

- `_config.yml` for site title, description, URL, and baseurl.
- `_data/novels.yml` for novel metadata and synopsis.
- `_data/support.yml` for Patreon link.
- `assets/covers/placeholder-cover.svg` with a real cover image.
- `_chapters/villainess-drinks-tea-alone/` with real chapter files.
- `chapters.json` when adding chapters, so previous/next navigation works.
